# EggSupplierSystem

A web project that helps deliver eggs from poultry to the suppliers, customers and keep the record of orders, deliveries, transactions. 

It consists of:
Frontend: HTML, CSS, and JavaScript for structure, styling, and interactivity.
Backend: Java with JSP (JavaServer Pages) for server-side logic and dynamic content generation.
Data Access: JDBC (Java Database Connectivity) to connect the application to the database.
Database: PostgreSQL, a relational database management system (RDBMS) for data storage.
